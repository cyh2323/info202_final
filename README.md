# Banking Product Recommender

A Streamlit app to explore and compare consumer banking products like credit cards, checking, and savings accounts. 
Users can filter products, use a Goal Discovery module, and compare up to 3 products side by side.

**Try it Online:** 
https://info202final-yonghyeok-choi.streamlit.app/

**Files:**
- `app.py` – Streamlit application
- `input_data.csv` – Sample dataset

**Run Locally:**
```bash
pip install streamlit pandas
streamlit run app.py
